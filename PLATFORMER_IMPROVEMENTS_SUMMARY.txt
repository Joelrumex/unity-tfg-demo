IMPLEMENTATION METHOD
================================================================================

Option 1: Using Helper Script (Recommended)
  1. Open Unity Editor
  2. Open Assets/Scenes/ForestPlatformerScene_Improved.unity
  3. Go to Window > Level Improvements > Add Enemies and Obstacles
  4. Script will automatically instantiate all enemies and obstacles
  5. Save scene (Ctrl+S)

Option 2: Manual Implementation via Inspector
  1. Open Assets/Scenes/ForestPlatformerScene_Improved.unity
  2. For each skeleton position:
     - Drag skeleton prefab into scene
     - Position at specified coordinates
     - Scale to 0.527 (match existing)
  3. For each spike position:
     - Duplicate existing Triangle instance
     - Move to new coordinates
  4. Save scene

Option 3: Direct YAML Editing
  1. Edit Assets/Scenes/ForestPlatformerScene_Improved.unity
  2. Add PrefabInstance entries for each enemy/obstacle
  3. Update SceneRoots at end of file with new fileID references
  4. Unity will recognize changes on reload
